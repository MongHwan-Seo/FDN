# FDN Reproducibility

This repository provides the data download script used in the FDN project.

## Data Source

Binance API

## Assets

BTCUSDT and ETHUSDT

## Data

* Spot OHLCV, 1h
* Perpetual futures OHLCV, 1h
* Funding rates

## Usage

```bash
python download.py
```

The downloaded files are saved in the `raw/` directory.
